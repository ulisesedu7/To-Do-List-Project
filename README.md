# To-Do-List

This repository contains an web app that allows to add and remove items to a to-do-list data using Javascript.

## Built With

- HTML
- SCSS
- Javascript
- Git & Github
- WebPack

## Live Demo

- Check the live demo of my page [here](https://ulisesedu7.github.io/To-Do-List-Project/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Modern web browser with Javascript enabled

Run: `Git Clone + 'URL of project'`
